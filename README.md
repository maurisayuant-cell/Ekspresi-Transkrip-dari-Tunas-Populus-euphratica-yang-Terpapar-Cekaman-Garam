# Ekspresi-Transkrip-dari-Tunas-Populus-euphratica-yang-Terpapar-Cekaman-Garam
Analisis ekspresi transkrip Populus euphratica di bawah stres garam menggunakan dataset publik GEO (GSE52305). Repository ini mencakup kode analisis DEG, visualisasi logFC, dan interpretasi gen yang terlibat dalam adaptasi terhadap cekaman salinitas, jalur stres osmotik, regulasi ion, dan pertahanan seluler.
